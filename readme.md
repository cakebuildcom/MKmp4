# MKmp4
Creates a MP4 from a WAV or MP3 file, also creates a dynamic image for the MP4.


## Windows batch file 
* Searches a folder **"source"** 
* Output goes into a folder **"output"**
* Moves the source file to a folder **"processed"**

> [!NOTE]
> Included the required folder you can delete **".gitkeep"** files.

## Requirements
* **Windows**
* **FFmpeg**:  In the same directory as the batch file.

## Output example
* All text can be changed in the batch file
* All info is created dynamically
  * Label of conversion "WAV to MP4" / "MP3 to MP4" 
  * Original file name
  * Original file size
  * Original file date
  * Encode date
